O que levar? personal_bag_question

  • Bíblia, caderno e caneta (para devocional e oficinas).
  • Documento de identificação (RG.).
  • Para higiene: sabonete, xampu, escova e creme dental, desodorante, protetor solar, repelente, pente, escova.
  • Para limpeza: sacos plásticos para roupas sujas e pano de chão para banheiro.
  • Para o quarto: cobertor, fronha, lençol (solteiro) e travesseiro.
  • Tolha de banho e rosto, toalha para piscina e toalha para sentar-se na grama.
  • Vestimentas: meias e roupas para calor, frio, piscina, gincanas.
  • Calçados: chinelo, tênis...

O que não levar?

  • Nenhum aparelho eletrônico ou sonoro:
    • Celular,
    • Ipad,
    • Iphone,
    • Notebook,
    • Tablet,
    • Vídeo Game...

Regras warning

  • Cumprir os horários estabelecidos da programação (acordar, dormir, refeições, cultos...).
  • Os objetos pessoais são de responsabilidade individual.
  • Vestimentas (cultos):
    • Para as meninas: não usar
      • vestido ou saia acima do joelho,
      • bermuda curta,
      • roupas transparentes (principalmente, brancas),
      • roupas justas e com decotes.
    • Dica: Caso use vestido ou saia, coloque uma bermuda segunda pele por baixo para evitar marcar qualquer parte do corpo.
    • Para os meninos: não usar
      • bermuda curta,
      • roupas transparentes (principalmente, brancas),
      • camiseta regata,
      • roupas justas.
  • Vestimentas (piscina):
    • Para meninas: maiô e shorts por cima (sem transparência).
    • Para meninos: bermuda e camiseta regata (sem transparência).
  • É proibido acessar os lugares sem autorização:
    • quartos de líderes,
    • cozinha,
    • meninos acessarem os quartos das meninas e vice-versa,
    • fazer barulho no horário de dormir.
